# FiscalPro BPO — Site Institucional

Site de apresentação do serviço de Entrada de Notas Terceirizada no SG Sistemas.

## 🌐 Domínio
`fiscalprobpo.com.br`

## 📁 Estrutura
```
fiscalprobop/
└── index.html   ← arquivo único do site
```

## 🚀 Deploy
Hospedado na [Vercel](https://vercel.com) com deploy automático a cada push no GitHub.

## ✏️ Como editar o site
Abra o `index.html` em qualquer editor de texto (VS Code, Notepad++) e edite:
- **WhatsApp**: busque `wa.me/` e troque o número
- **E-mail**: busque `contato@fiscalprobpo.com.br` e troque
- **Preços**: busque `R$` e ajuste os valores
- **Textos**: edite diretamente o conteúdo das seções

Após salvar, faça commit e push — o site atualiza automaticamente em ~30 segundos.
